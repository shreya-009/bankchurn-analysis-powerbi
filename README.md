# 📊 Bank Churn Analysis (Power BI)

## 📌 Project Overview

This project analyzes customer churn behavior in a bank using Power BI.
The goal is to identify key drivers of churn and provide actionable insights for customer retention.

---
## 🎯 Objectives

Identify high-risk customer segments
Analyze churn across demographics and behavior
Understand root causes of customer churn

---
## 📁 Dashboard Structure

🔹**Executive Summary**
- Total customers, churn rate, active vs inactive
- Churn distribution by gender, country, and credit rating

🔹**Customer Segmentation**
- Churn by age group
- Churn by number of products
- Churn by credit card ownership
- Trend analysis over time

🔹**Churn Drivers** (Deep Dive)
- High-risk segment: older inactive customers
- Product usage vs churn behavior
- Balance-based churn segmentation
- Detailed churned customer table

---
## 📈 Key Insights
1. Older (50+) inactive customers show highest churn
2. Customers with 3 products have significantly higher churn
3. Low balance customers are more likely to churn
4. Non-credit card holders show slightly higher churn
5. Customer inactivity is a major driver of churn

---
## 🛠 Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning
